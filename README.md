## About
Nexora is a modern real-time communication platform inspired by Discord, focused on performance, modern UI and scalable architecture.

## Features
- Real-time messaging
- Authentication system
- Server and channels
- Online users
- Dark mode
- Responsive UI

## 🛠 Technologies

### Front-end
- React
- JavaScript
- CSS

### Back-end
- Python
- FastAPI

### Database
- MySQL

### Real-Time Communication
- WebSocket
